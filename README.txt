OpenSCAD with Preview + Camera Gizmo

Includes:
- Code editor with syntax highlight
- 3D preview pane with orbit controls and axis gizmo
- Library insert dropdown
- Local file library

Libraries are NOT bundled due to size (BOSL2 = 15MB). To add them:
1. Download from GitHub:
   - BOSL2: https://github.com/BelfrySCAD/BOSL2/archive/master.zip
   - MCAD: https://github.com/openscad/MCAD/archive/master.zip
   - Round-Anything, NopSCADlib, etc.
2. Unzip, then use Import button in app to add .scad files
3. They will be saved in browser storage

For full offline bundle, download libraries once on computer, then import.
